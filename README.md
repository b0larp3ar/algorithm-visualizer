# Algorithm Visualizer

An interactive web-based visualizer for sorting algorithms.

## Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

## Features
- Adjustable speed
- Adjustable array size
- Clean UI with control panel

- HTML
